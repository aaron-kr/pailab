## Who it's for

Database design or security courses. Students already know SQL — this module
bridges to time-series data typical in Physical AI pipelines.

## Schedule (2 weeks)

**Week 1** — Why relational DBs struggle with sensor streams; InfluxDB line protocol; Telegraf agents  
**Week 2** — Grafana dashboards; retention policies; anomaly detection query patterns; project: live sensor dashboard from an Arduino/RPi source
