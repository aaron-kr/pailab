[Our paper](https://doi.org/10.56977/jicce.2024.22.1.80) demonstrated that it is possible to semi-manually gather ancient Manchu script letters using OpenCV's `selectROIs()` function. [A follow-up conference paper](https://www.researchgate.net/publication/392027771_Development_of_an_Interactive_Dataset_Builder_for_Ancient_Script_Recognition_Using_ROI_Selection_and_Template_Matching) demonstrated that a Python-based Desktop application speeds and improves collection of manually hand-selected script letters. This research will be used to develop a full and comprehensive dataset handwritten Cherokee letters.
